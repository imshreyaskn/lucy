How to use  Lucy

Three simple steps to navigate the web at the speed of thought.

01

Invoke

Press Cmd/Ctrl + Shift + K or click the extension icon to wake Lucy up.

02

Command

Speak naturally. Ask Lucy to summarize an article, find information, or fill out a form.

03

Relax

Lucy autonomously navigates the DOM, clicks buttons, and gets the job done for you.

Install Lucy

Your download should start automatically. Follow these steps to install Lucy via Developer Mode:

  1. Extract the downloaded lucy-v1.0.1.zip file.
  2. Open a new tab and go to chrome://extensions/
  3. Turn on Developer mode in the top right corner.
  4. Click Load unpacked and select the extracted folder.
